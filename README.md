## Funcionalidades

- #### Chatea sin registro.
- #### Contador de usuarios de cada grupo
- #### Diseño adaptativo para cualquier plataforma.
- #### Envia imagenes
- #### Formatea enlaces URL para ingresar con un solo click

## Instalación

## .env 
```
NEXT_PUBLIC_SERVER_URL
```

- ### Requiere Anonytalk_Server para funcionar

```sh
cd anonytalk_web
npm i
npm run dev
```
